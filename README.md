# shopadmin
> &nbsp; shopadmin 是一个以VUE为前端框架Element-ui为UI框架的后台管理项目，
> 项目目前处于开发阶段。

### 项目集成：

![123.png](https://i.postimg.cc/prCddy9P/123.png?raw=true)

> 网站依赖Element-ui封装layout布局：
> - 侧边栏
>> 根据路由文件自动生成侧边栏。 <br/>
>> 侧边栏集成权限配置、文字图标、汉堡菜单
> - 头部导航栏
>> 导航栏集成面包屑、用户信息
> - 主内容区
>> 主内容区域依赖侧边栏、头部导航路由跳转及响应页面。 <br/>
>> 首页仪表盘集成数字滚动、饼状图、柱状图、线型图。
